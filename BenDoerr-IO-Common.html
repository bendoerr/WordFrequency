Dealing with IO errors

ioHandled :: IO () -> IO ()Source

Catch IOErrors using the commonErrorHandler.

Use on main functions for consistent handling of IOErrors at the top level. 

 main = ioHandled $ do
                    c <- getContents
                    putStrLn c

commonErrorHandler :: IOError -> IO ()Source

Handle top level IOErrors consistently.

  • Currently only handles DoesNotExistErrors.
  • Will continue to add as I run into them.

readFromStdinOrArgsFiles :: IO ByteStringSource

Wrapper around readFromStdinOrFiles that passes arguments from getArgs.

readFromStdinOrFiles :: [FilePath] -> IO ByteStringSource

If an empty list is given then read from Stdin. Otherwise try reading from files specified by each of the argument list, then join them togeather.

This should read multiple files using near constant memory, there will be a little bump for each addtional file due to an additional file handle.
